# Video Notes AI UI v7

## 目标

UI v7 依据 Windows 实机 1650×900 截图重新校准，不再追求“高密度后台面板”，而以可读、可操作、适合长时间使用的桌面生产力工具为目标。

## 核心修复

- 移除 7–11px 的微型文字体系，正文与操作文字提升至 13–16px。
- 左侧栏扩展为 248px，导航、品牌、引擎状态与主操作按钮重新分级。
- 顶部栏改为轻量面包屑与状态工具栏，避免与页面标题重复。
- 页面内容宽度限制在 1240px，防止超宽窗口把信息拉散。
- 任务中心重新设计统计卡、筛选栏、任务表格、空状态与详情面板。
- 创建笔记页强化三步流程、输入区、模型区和当前任务摘要。
- 笔记库提升列表、阅读器和 Markdown 正文字号。
- 合集与设置页重新调整导航宽度、表单密度、卡片尺寸和弹窗可读性。
- 对 1180–1280px 窗口增加响应式降级，避免列宽挤压。

## 适配基线

- 1366×768：保持可操作，不出现横向挤压。
- 1650×900：主要设计基线。
- 1920×1080：内容居中，不无限拉宽。
- Windows 125%/150% 缩放：依赖 WebView CSS 像素，不再使用小于 11px 的界面文字。
